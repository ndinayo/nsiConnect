# nsiConneKt

Professional staffing website for Rwanda — chefs, housekeepers, nannies, and security.

## Local development

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this repo to [GitHub — ndinayo/nsiConnect](https://github.com/ndinayo/nsiConnect).
2. Go to [vercel.com/new](https://vercel.com/new) and import **ndinayo/nsiConnect**.
3. Vercel auto-detects **Vite** — keep defaults:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
4. Click **Deploy**.

No environment variables are required for the static site.
